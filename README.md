
  # Smart Safety Care 


🛡️ Smart Safety Care：您的 24 小時全方位 AI 智能照護管家
Smart Safety Care 是一款結合了尖端 AI 影像辨識、健康數據追蹤與日常照護管理的智慧型應用程式。無論您身在何處，都能透過手機即時掌握被照護者的安全與健康狀態，讓科技為家人築起最安心的防護網。
🌟 核心功能亮點
1. 👁️ AI 智能監控與風險預警 (監控中心)
即時動態掌握：支援多頻道串流，可隨時查看家中各個角落（如客廳 CAM-01）的即時影像，並提供歷史影像回放功能
。
即時風險評估中心：獨家 AI 技術能針對畫面進行即時動態評分，精準計算目標的「距離（近度）」、「危險等級」與「持續時間」
。一旦偵測到潛在危險（如接觸危險刀具），系統將立即啟動防護機制。
首頁捷徑整合：在監控首頁即可看見即將到來的照護任務（如「午餐均衡飲食」），讓監控與照護無縫接軌
。
2. ❤️ 醫療級健康數據看板 (健康管理)
五大生理指標實時追蹤：一目了然的儀表板，為您即時呈現「實時心電圖 (ECG)」、「心率」、「血壓」、「血氧 (SpO2)」及「體溫」等關鍵數據
。
AI 智慧分析與總評：系統會為每項指標提供狀態評級（如：正常、優良），並自動結算「今日健康總評」分數，透過「智慧分析」功能協助照護者快速掌握長輩的身體狀況
。
3. 📋 智慧照護日誌與任務追蹤 (照護管家)
待辦任務清單：協助您井然有序地管理每日照護進度（如：服用降血壓藥物、回報血壓數值、低鹽低糖飲食、飯後散步），並有清晰的進度條顯示當日完成度
。
全天候事件日誌：詳實記錄所有異常與活動，支援標籤分類（警報、用藥、人員）。無論是「疑似火源」的危險警報，或是「陌生訪客進入玄關」的人員偵測，通通不漏接
。
AI 今日生活觀察總結：每日自動生成生活報告，統整被照護者的步態指標、用藥狀況與整體防護狀態，大幅減輕照護者的紀錄負擔
。
4. ⚙️ 高度客製化與軍規級安全 (系統設置)
強大且彈性的 AI 引擎：系統底層採用先進的 YOLOv8-PyTorch 核心引擎，可自訂 YOLO 辨識模型（如邊緣運算專用的 best_fp16.tflite），並允許使用者自訂「警報觸發閾值」與設定 n8n 通知掛鉤，實現高度客製化的自動化通報流程
。
影音設備擴充：支援輸入 RTSP 網址無縫連接各種 WiFi 攝影機，並可自由調整影像畫質、開啟現場收音及「雙向通話」功能，讓遠端關懷零距離
。
隱私安全最高標準：全面啟用「系統安全性」，主打本機運算與端對端加密傳輸，確保所有極機密的生理數據與居家監控影像絕對不外洩，保護家人的隱私權益
。
Smart Safety Care 不僅僅是一個監控鏡頭，更是結合了安全防護、健康監測與生活管理的三合一智慧大腦，為您的居家照護提供最強大的科技後盾！
