
  # Smart Safety Care App

Smart Safety Care 
■ 產品概述
Smart Safety Care 是一款結合 AI 影像辨識與即時監控的照護系統。透過電腦視覺技術，系統能 24 小時不間斷地識別環境中的人體動態與特定危險物品，在意外發生前為照護者提供關鍵預警。

■ 核心功能說明
24h 人體動態偵測：系統持續掃描監控畫面，精確鎖定人體位置並追蹤其活動狀態，確保照護對象始終在安全感測範圍內。

特定危險物判別（刀具/剪刀）：

精準辨識：內建專門針對「刀」與「剪刀」訓練的 AI 模型。

主動預警：當監控範圍內出現上述危險尖銳物品，且人體靠近或接觸時，系統將即時發出判別訊號，防止割傷或意外發生。
